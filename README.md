# README

### Before Running

* `bundle install`
* `rake db:create`
* `rake db:seed`
* `bin/vite update`
* `yarn`

### Running (local)

* `bin/vite dev`
* `bundle exec rails s`

### Testing

* `bundle exec rspec`
