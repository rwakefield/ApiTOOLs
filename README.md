# README

### Before Running

* `bundle install`
* `rake db:create`
* `rake db:seed`
* `rake rswag:specs:swaggerize`
* `bin/vite update`
* `yarn`

### Running (local)

* `bin/vite dev`
* `bundle exec rails s`

### Testing

* `bundle exec rspec`
